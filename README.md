# Apex Fitness Club Website

A static landing page for the Apex Fitness Club gym, built with HTML, CSS, and JavaScript. The site showcases gym facilities, services, membership plans, and a contact form.

## Features

- Responsive one-page layout with anchored navigation
- Hero section with gym branding and call-to-action
- Facility and services sections with visual cards
- Membership plan comparison
- Contact form with client-side validation and submission feedback
- "Read More" content reveal interaction

## Project Structure

- `index.html` – main page markup and section structure
- `styles.css` – page styling, layout, responsive design, colors, and buttons
- `script.js` – interactive behavior for form submission and read-more toggle
- `Images/` – image assets used throughout the landing page

## How to Use

1. Open `index.html` in your browser.
2. Navigate using the top nav links or page sections.
3. Click the "Read More" button in the About section to expand extra content.
4. Fill out the Contact form and submit to see the client-side validation message.

## Notes

- This is a static site, so there is no backend form processing.
- The contact form currently displays a success message locally and resets the fields.
- Update the file paths in `index.html` if you move image assets or CSS/JS files.
